# design-system-starter

Template monorepo pour industrialiser un Design System from scratch.

## Prérequis

- Node 18+
- pnpm 8+
- Git

## Démarrage

```bash
git clone https://github.com/RomainRichardpro/design-system-starter.git
cd design-system-starter
make setup
make dev
```

Ouvre http://localhost:6006 — le guide complet est dans Storybook.
