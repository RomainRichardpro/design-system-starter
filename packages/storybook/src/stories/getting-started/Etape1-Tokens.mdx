# Étape 1 — Intégrer tes tokens Figma

Les tokens sont les valeurs fondamentales de ton Design System :
couleurs, typographie, espacements, rayons de bordure.

Cette étape transforme tes variables Figma en variables CSS
utilisables dans tous tes composants.

## 1. Exporter tes variables depuis Figma

Dans Figma :

1. Ouvre le panneau **Assets** (icône grille en haut à gauche)
2. Clique sur **"Variables"**
3. Clique sur l'icône d'export en haut à droite du panneau
4. Choisis le format **JSON**
5. Télécharge le fichier

> Le fichier téléchargé contient toutes tes variables Figma
> dans un format que Style Dictionary peut lire.

## 2. Placer le fichier dans le projet

Remplace le fichier vide par ton export :

```
packages/tokens/tokens.json  ← remplace ce fichier par ton export Figma
```

## 3. Adapter le mapping à tes collections

Ouvre `packages/tokens/mapping.config.js`.

Ce fichier fait le lien entre tes collections Figma et les fichiers CSS générés.
Tu dois adapter les valeurs `figmaCollection` pour qu'elles correspondent
exactement aux noms de tes collections dans Figma.

Pour trouver le nom exact d'une collection dans Figma :
1. Ouvre le panneau Variables dans Figma
2. Le nom de la collection apparaît en haut de chaque groupe
3. Reporte ce nom exactement dans `mapping.config.js`
   (attention aux majuscules et aux espaces)

Exemple :

```js
export default {
  collections: [
    {
      figmaCollection: "Couleurs/Light", // ← nom exact dans Figma
      output: "colors-light.css",
      type: "color",
      mode: "light",
    },
    // Ajoute autant d'entrées que tu as de collections
  ]
}
```

## 4. Lancer le build des tokens

```bash
make tokens
```

## 5. Vérifier dans Storybook

Navigue dans **Fondations** dans le menu à gauche.

✅ Tu vois tes couleurs, ta typographie et tes espacements ?
Les tokens sont correctement intégrés.
Continue avec **Étape 2 — Premier composant**.
