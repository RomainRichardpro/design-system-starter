# Bienvenue dans ton Design System

Tu as réussi l'installation — Storybook tourne sur ton ordinateur.
À partir d'ici, tout se passe dans ce guide.

## Ce que tu vas construire

- Tes tokens Figma transformés en variables CSS utilisables dans le code
- Des composants React accessibles, testés et documentés
- Une documentation Storybook déployée en ligne automatiquement
- Une connexion directe entre Figma et le code via Code Connect

## Comment utiliser ce guide

Le guide est organisé en 4 étapes dans le menu à gauche.
Suis-les dans l'ordre — chaque étape suppose que la précédente est faite.

- **Étape 1 — Tokens Figma** — Exporter et intégrer tes variables Figma
- **Étape 2 — Premier composant** — Créer un composant avec Claude AI
- **Étape 3 — CI-CD** — Automatiser les tests et le déploiement
- **Étape 4 — Code Connect** — Relier Figma et le code
