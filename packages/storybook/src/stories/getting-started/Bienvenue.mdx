# Bienvenue dans ton Design System

Ce template te permet de construire un Design System industriel,
même sans expérience en développement.

Il repose sur les mêmes outils que les équipes produit les plus avancées :
Figma comme source de vérité, React pour les composants,
Storybook pour la documentation, et Claude AI pour accélérer la génération de code.

## Ce que tu vas construire

- Tes tokens Figma transformés en variables CSS
- Des composants React accessibles et testés
- Une documentation Storybook déployée en ligne
- Une connexion directe entre Figma et le code (Code Connect)

## Auteur

Ce template a été conçu par **Romain Richard**, Lead System Designer,
avec la conviction qu'un bon Design System se construit avec méthode,
pas avec des raccourcis.
