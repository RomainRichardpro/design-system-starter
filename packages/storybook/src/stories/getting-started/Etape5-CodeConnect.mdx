# Étape 5 — Code Connect

Code Connect lie tes composants React à leurs contreparties dans Figma.
Il permet à Figma de proposer le bon code lors du handoff.

## Prérequis

- Composants React stables (étape 3 complétée)
- Figma avec les composants correspondants publiés

## Configuration

_Le contenu détaillé de cette étape sera ajouté en Session 4._

## Principe général

Pour chaque composant, tu crées un fichier `.figma.tsx` qui mappe :
- Le node ID Figma du composant
- Les props React correspondantes
- Les variants Figma → valeurs de props

---

✅ **Critère de succès** : Le composant affiche du code React dans le panneau Dev de Figma.
