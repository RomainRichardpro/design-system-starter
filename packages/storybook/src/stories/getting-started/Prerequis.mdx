# Prérequis

Avant de commencer, vérifie que ton environnement est correctement configuré.

## Outils requis

- **Node.js** 18 ou supérieur
- **pnpm** 8 ou supérieur
- **Git**
- **Figma** avec accès aux Variables du fichier source

## Vérification

```bash
node --version   # doit afficher v18.x ou supérieur
pnpm --version   # doit afficher 8.x ou supérieur
git --version
```

## Installation de pnpm (si nécessaire)

```bash
npm install -g pnpm
```

---

✅ **Critère de succès** : Les trois commandes s'exécutent sans erreur.
