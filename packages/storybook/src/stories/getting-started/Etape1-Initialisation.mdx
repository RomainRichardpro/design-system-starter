# Étape 1 — Initialiser le projet

## 1. Copier le template

Sur la page GitHub du template, clique sur **"Use this template"**
puis **"Create a new repository"**.

Donne un nom à ton repo, laisse les options par défaut,
et clique sur **"Create repository"**.

## 2. Cloner le repo sur ton ordinateur

Ouvre ton terminal et tape ces commandes
(remplace l'URL par celle de ton repo) :

```bash
git clone https://github.com/TON-COMPTE/TON-REPO.git
cd TON-REPO
```

## 3. Installer les dépendances

```bash
make setup
```

> Cette commande peut prendre 1 à 2 minutes.
> Tu verras défiler des lignes dans le terminal — c'est normal.

## 4. Lancer Storybook

```bash
make dev
```

Ouvre ton navigateur sur **http://localhost:6007**

✅ Tu vois cette page ? L'installation est réussie.
Continue avec **Étape 2 — Tokens**.
