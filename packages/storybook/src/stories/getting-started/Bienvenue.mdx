# Bienvenue dans le Design System Starter

Ce template est une infrastructure monorepo prête à l'emploi pour industrialiser
un Design System from scratch sur n'importe quel projet client.

Il ne contient aucun token, aucun composant — c'est une base propre que tu vas
construire étape par étape.

## Ce que tu vas construire

1. **Tokens** — exporter tes variables Figma et les compiler en CSS
2. **Composants** — créer tes premiers composants React documentés
3. **CI/CD** — automatiser les tests et le déploiement
4. **Code Connect** — lier tes composants à Figma

## Comment naviguer ce guide

Suis les étapes dans l'ordre dans la sidebar :

- **Prérequis** → vérifie ton environnement
- **Étape 1** → initialisation du projet
- **Étape 2** → import des tokens Figma
- **Étape 3** → premier composant
- **Étape 4** → CI/CD
- **Étape 5** → Code Connect
